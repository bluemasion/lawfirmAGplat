# 中国移动采购投标文件模板结构

> 来源：客户提供的招投标文件.docx (2026-02-26)

## 文档整体结构

### (一) 报价分册
1. **投标一览表** — 项目编号、投标人名称、采购内容、数量、纳税人性质、未含税总价、税率、含税总价
2. **分项报价表** — 根据项目实际情况编制
3. **招标代理服务费支付同意函** — 差额定率累进法计算

### (二) 商务分册
1. **评标索引表** — 初步评审 + 详细评审
2. **投标函** — 致招标人的正式承诺函（7项承诺）
3. **资格审查资料**
   - 3.1 基本情况表（名称、注册资本、成立时间、地址、联系方式、法人）
   - 3.2 营业执照/组织机构代码证/税务登记证
   - 3.3 增值税一般纳税人证明材料
   - 3.4 授权条件（制造商资格声明、授权函、代理商声明）
   - 3.5 控股及管理关系情况申报表
   - 3.6 非联合体投标及不存在转包承诺函
   - 3.7 企业信誉声明函
   - 3.8 中小微企业声明函
4. **商务条款偏离表** — 11项条款逐一响应
5. **投标保证金** — 电汇/银行保函
6. **业绩情况表** — 近年类似项目
7. **财务状况** — 总资产/负债/利税/营收
8. **其他证明资料**

### (四) 法人授权书
1. **法定代表人身份证明**
2. **法定代表人授权委托书**

## AI 可自动生成的内容（框架搭建）

| 章节 | AI 生成方式 | 需要的输入 |
|------|-----------|-----------|
| 投标函 | 模板填充 + 措辞优化 | 投标人名称、项目名称、招标人名称、保证金金额 |
| 基本情况表 | 结构化填充 | 公司名称、注册资本、成立时间、地址、联系人 |
| 投标一览表 | 结构化填充 | 采购内容、数量、报价、税率 |
| 企业信誉声明函 | 模板生成 | 投标人名称、项目名称 |
| 非联合体承诺函 | 模板生成 | 投标人名称、项目名称 |
| 控股关系申报表 | 结构化填充 | 股东信息 |
| 商务条款偏离表 | 默认"全部无偏离" | 无 |
| 法人授权委托书 | 模板填充 | 法人姓名、委托人姓名、项目名称 |

- 业绩合同扫描件
- 银行保函/保证金凭证
- 增值税纳税人证明
